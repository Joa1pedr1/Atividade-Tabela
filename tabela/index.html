Tabela de Linguagens - JP

Linguagem Opinião
Python Python é uma linguagem de alto nível, muito utilizada em projetos que envolvem Machine Learning. O ponto alto desta linguagem é que ela é ótima como porto de partida para o mundo da tecnologia, por ser intuitiva e objetiva.
Javascript JavaScript é uma linguagem de script orientada a objetos e plataforma. É responsável por tornar as páginas web mais interativas com o usuário. Além disso, é muito utilizada em desenvolvimento de jogos também.
Java Esta linguagem possui uma ampla variedade de possibilidades. Ela pode ser responsável para desenvolvimento de software (desktop e mobile), de senvolvimento web, desenvolvimento de jogos, IoT e outras.
C# Linguagem desenvolvida pela Microsoft se configura, também, como uma linguagem bem versátil. Os programadores desta linguagem desenvolvem aplicativos para Windows da Windows Store e jogos com a ajuda da Unity Engine.
PHP Esta linguagem é extremamente usada para desenvolvimento Web e pode ser incorporado diretamente ao HTML para facilitar a criação de páginas.
Ruby Essa linguagem foi criada no Japão e é conhecida por sua programação dinâmica. A Ruby é uma linguagem orientada a objetos e tem, como uma de suas principais funções, também, o desenvolvimento web.
